# zetton-common

Common definitions and utilities in ROS environment for Project Zetton.

## Prerequisites

Recommended environment:

- Ubuntu 18.04
- ROS Melodic
- OpenCV 4.1.0+
- GStreamer 1.14+

## Usage

### RTSP stream

Receive RTSP stream by OpenCV + GStreamer

- Choose `avdec_h264` for PC and `omxh264dec` for Jetson devices
